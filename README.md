# RNA-seq Differential Expression Analysis (Airway Dataset)

## Overview
This project demonstrates end-to-end RNA-seq differential expression analysis
using the Bioconductor airway dataset.

## Workflow
- Load RNA-seq count data
- Differential expression analysis using DESeq2
- Quality control & visualization
- GO and KEGG pathway enrichment

## Tools & Packages
- R / Bioconductor
- ExperimentHub, airway
- DESeq2, SummarizedExperiment
- EnhancedVolcano, pheatmap
- clusterProfiler, enrichplot
- org.Hs.eg.db, AnnotationDbi
